# 【锁定】尺子思维：丈量问题的边界
> [!IMPORTANT] 锁定: 发现问题苗头，立刻锁定
- 认知问题
	- 问题的分类
		- 日常问题: 有标准答案
		- 难题: 需要提出创造性的解决方案
- 确定问题改善之处
	- 对问题进行深入的分析
	- 确定改进方案
	- 对方案进行落地
- 解决问题的路径
	- 路径1：过去的再现化->解决日常问题
		- 对现有问题找到正确的答案
	- 路径2:  未来的具象化->解决非常规问题
		- 不只是解决现有的某一个问题
		- 是对现有问题的体系化解决
		- 且为了防劣化，对该类问题制定一系列的规范
- 确定解决问题的方法
	- 关键步骤: 
		- 分析 
			- 信息收集
			- 功能整理
			- 问题分析
		- 创构 
			- 捕捉灵感
			- 创作
			- 构思
		- 锤炼
			- 利弊分析
			- 缺点筛查
			- 效果评估
- 会解决问题 VS 不会解决问题
	- 会解决问题
		- 难题
		- 思考、思考、思考
		- 解决问题后的理想状态
		- 最终制定解决方案
	- 不会解决问题
		- 调查、调查、调查
		- 在现有架构的基础上，无法真正彻底的解决问题的
		- 一直致力于寻求正确答案
		- 无法提炼出一个完备的方案
# 【分析】剪刀思维：分割杂糅问题
> [!IMPORTANT] 分析
> 1. 分解+解析
> 2. 先拆解再整合
> 3. 颠覆看待问题的角度

- 明确问题及对象
	- 将问题大卸八块后，重新建构
		- 将问题拆分成最小的单位
		- 重新组合
		- 整理、汇总
- 重新定位解题思路
	- 提供一套新的方案，实现原有功能
		- 抛开束缚
		- 重新组织
		- 聚焦功能
# 【创构】针线思维：链接碎片化问题
> [!IMPORTANT] 创构
> 1. 将碎片化的想法重组
> 2. 创新的方案
- 构思的工具
	- 头脑风暴法
		- 追求数量
		- 严禁评判
		- 自由奔放
		- 与改善想法相结合
	- 类比思考法
		- 相似问题和经验的类比
	- 检核表法
		- 是否有替代品
		- 是否可增大、减少
		- 是否可以删除
		- ...
# 【锤炼】锤子思维：”实锤“疏忽问题
> [!IMPORTANT] 锤炼: 方案->审视、推敲->方案

- 循环锤炼方案`
	- 寻找缺点 -> 克服缺点 -> 继续进行审视
	- 最终获得一个完善的方案
- 制作潜在的缺点清单
	- ① 否定最初的构想
		- 不断地挑剔细节
		- 寻找疏漏
	- ② 全方位审视问题
		- 多视角审视
			- 尽可能接近问题本质
		- 多立场审视
			- 综合客观的审视问题
	- ③ 制作潜在缺点清单
		- 罗列构想和观点
		- 清晰问题视角
		- 逐一核查
- 思考其他可能性
	- 保持辩证的态度
		- 搜寻缺点(批判)
		- 剔除缺点(肯定)
	- 重新构思方案
		- ① 构思解决问题的框架
		- ② 深入构想内部
		- ③ 思考其他可能性
		- ④ 分类整合
		- ⑤ 在”锤炼循环“中检验和完善
		- ⑥ 测试和检验问题
		- ⑦ 听取他人意见
		- ⑧ 克服缺点
- 规避思维陷阱
	- 陷阱
		- 近乎完美的方案
		- 但事态发展偏移原本的预期
			- 过于复杂
			- 成本过高
			- 难以实现
			- ...
	- 飞石法则: 必须迎难而上
		- 选容易踩到的石块->无法到达对岸->无法真正解决问题
		- 选有挑战的石块->到达对岸->彻底解决问题
- 评价的可视化
	- ① 明确可靠性和现实性-> 实施前后的投资/成本/效果差异
	- ② 对解决方案的评价-> 分析数据->判断价值
	- ③ 绘制评价图->实现评价"可视化"
# 【完善】螺丝刀思维：拧紧大脑的发条，让拆解成为一种习惯
>[!IMPORTANT] 完善: 
>1. 改善解决方案的缺陷与不足
>2. 预想实施阶段的潜在障碍，并彻底消除

- 刻意练习
	- 解决问题的技能
		- 构思
			- 分析->创构->锤炼
		- 完善
			- 预想障碍、彻底消除
		- 应用
			- 将方案运用到解决实际问题中
	- 习得性的拆解问题
		- ① 提问”为了谁？ 为什么？“ -> 提高问题的认识能力
		- ② 分解问题并分析结构后的问题-> 提升确定问题改善点的能力
		- ③ 提出解决问题的独特构想-> 提升构思/创造能力、想象力
		- ④ 剔除现有方案缺点、并尝试克服
		- ⑤ 剔除自己的意见、建议、疑问
- 培养能力
	- 管理: 在有限制的情况下完成任务
		- 时间管理
			- 应对拖延症的能力
		- 成本管理
			- 成本管理的意识
		- 成果管理
			- 最大限度的达成目标
	- 团队: 带动人员和组织
		- 发挥团队成员的潜在能力
		- 将成员的经验和知识联系起来
		- 说服相关人员进行积极的实践活动
		- 思诺特模型
			- 理念+共识+能力+报酬+资源+实施计划=达成目标
	- 能力
		- 管理能力
			- 时间管理
			- 成本管理
			- 成果管理
		- 协调能力
			- 业务协调
			- 人际关系协调
			- 组织协调
		- 执行能力
			- 知识
			- 毅力
			- 体力
		- 达成能力
			- 策划
			- 制作资料
			- 应对质疑
- 训练思维
	- 锻炼脑部肌肉
		- 平时努力
			- 有计划的练习
			- 努力解决问题的意思
			- 行动并坚持到底
		- 肌肉训练
			- 伸展肌肉
				- 提出想法
				- 每条坚持提出简单的想法
			- 锻炼肌肉
				- 解决问题
				- 埋头苦干，付诸行动
		- ”1.01法则“和”0.99法则“
			- 含义
