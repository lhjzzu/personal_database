# 什么是能力圈？
- 你不需要成为一个通晓所有行业的专家。你只需要能够评估在你能力圈范
围内的公司就足够了。
- 能力圈的大小并不重要，重要的是，你要清楚自己的能力圈边界在哪
# 跳出安全圈的心魔?
1. **焦虑** : 比较式焦虑。<font color='red'>让你看不清自己</font>
2. **贪婪**: 不仅指对金钱的贪婪。更指对“捷径”的贪婪。超高的回报伴随着超高的风险。 <font color='red'>让你看不见风险</font>
3. **傲慢**: 错误的将自身特定领域的能力，泛化的认为是全领域的能力  <font color='red'>让你看不到边界</font>
# 如何降服心魔?
1. 向内挖掘，找到你的“热情矿脉” (<font color='red'>热情清单</font>)
2. 全面盘点，绘制技能图谱 (<font color='red'>能力清单</font>)
3. 交叉验证，锁定能力核心区 (<font color='red'>交集</font>)

# 总结
- 要找到热情和能力的交集，即<font color='red'>能力的核心区</font>
